## 环境准备

- **SDK**: [.NET 8.0](https://dotnet.microsoft.com/download) 或更高版本
- **IDE**: Visual Studio 2022 / VS Code + C# 扩展
- **数据库**: SQL Server / PostgreSQL / SQLite（按需选择）
- **包管理**: NuGet

```bash
# 验证安装
dotnet --version

# 创建新项目
dotnet new console -n MyFirstApp
dotnet run
```
